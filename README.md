# NewIDC，新一代域名主机管理系统

## About NewIDC

NewIDC 是一款类似WHMCS的域名主机兼财务管理系统，主要的功能为对接虚拟主机/VPS/服务器、自动化开通服务、自助充值和账单管理等，
功能强大，支持插件和主题更换（doing），且完全开源。

## 已实现功能

* 服务自动开通/到期暂停/销毁
* 免费续期限制
* 多周期付费选择（待完善）
* 工单系统
* 邮件提醒
* 插件系统
* 用户管理
